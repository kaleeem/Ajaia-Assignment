# Ajaia Docs

A lightweight collaborative document editor built for the Ajaia AI-Native Full Stack Developer Assessment.

## Setup

```bash
pnpm i
pnpm run db:push

# Run both frontend and backend
pnpm dev
```

## Tech Stack

### Backend
- Next.js API Routes
- tRPC

### Frontend
- Next.js (App Router)
- Headless UI
- Tiptap Editor
